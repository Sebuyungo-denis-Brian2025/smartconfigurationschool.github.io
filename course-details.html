Front End Web application Developer

This course is meant for students who want to become Front end developers.The students will concetrate on designing user interfaces using HTML,CSS,SCSS vanilla javascript,fetch technologies like e.g Axios, fetch API.Frame works like React and libraries like boostrap will be at the center of the training.Through the developed interfaces they will also learn to interact with public APIs to create applications.The student can decide to become a full stack developer by learning API development in addition to front end development.
